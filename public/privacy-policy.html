Politica de Privacidade

Ultima atualizacao: 2026-03-25

1. Visao geral

Esta politica descreve como o FitLocal processa dados para oferecer funcionalidades de acompanhamento fitness.

2. Dados processados

  • Dados de perfil (ex.: nome, idade, peso, altura, objetivo).
  • Historico de check-ins e peso.
  • Fotos de progresso e notas opcionais.

3. Armazenamento

Os dados podem ser armazenados localmente no dispositivo/navegador para funcionamento do app.

4. Exclusao de dados

O usuario pode remover os dados locais diretamente no aplicativo usando a opcao Delete All Data.

5. Contato

Para solicitacoes de privacidade e suporte, use o contato informado no app.
